# Skyscope

# 프로젝트 소개

현재 내가 있는 장소, 지도로 검색한 장소의 현재 기온과 강수 확률에 대한 3일까지의 정보를 제공합니다.

공공 데이터 포털에서 제공하는 기상청 API와 카카오 지도 API를 사용했습니다.
